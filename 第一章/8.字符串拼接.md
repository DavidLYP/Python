# 字符串拼接
### 很多时候，我们需要将多个字符串拼接起来，可以直接使用“+”进行拼接,方式如下:

1. 多个字符串字面量直接书写
```
slogan = "黑马程序员""成就IT黑马"
print(slogan)
```
中间无论多少空格,输出仍旧是连在一起的



2.+号拼接
```
slogan = "黑马程序员"+"成就IT黑马"
print(slogan)

s1 = "人生苦短"
s2 = "我用Python"
print("吉多·范罗苏姆："+s1+","+s2)
```
 +号可以用来拼接两个字符串,但是无法将非字符串与字符串进行拼接(非字符串类型需要转换为字符串类型)
